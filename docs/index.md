# mZen
移动手机端CSS组件库

## 概序
mZen为移动手机端，网页界面快熟开发而定义的一套CSS库。

### 页面模版
＊ HTML5文档类型
＊ 更好手机端体验

基于以上两点，每个页面参照如下格式设置：

``` html
<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
<meta name="viewport" content="maximum-scale=1.0,minimum-scale=1.0,user-scalable=0,width=device-width,initial-scale=1.0" />
</head>
  ......
</html>
```
